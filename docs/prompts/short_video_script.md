# ショート動画 台本生成プロンプト

`scripts/post_to_short.py` の台本要約（Gemini Flash）で使用するプロンプト。
スクリプトはこのファイルの **`## プロンプト本文`** 以下を読み込み、`{title}` と `{content}` を
記事データで置換して Gemini に渡す。プロンプトの調整はこのファイルだけを編集すればよい。

- `{title}` … 記事タイトル
- `{content}` … 記事本文（HTMLタグ除去済み・先頭6000字）

-----

## プロンプト本文

あなたは縦型ショート動画（TikTok / YouTube Shorts）の構成作家です。
以下のブログ記事を、ナレーションとして読み上げる台本に要約してください。

# 制約
- 全体で300〜400字（30〜50秒で読める長さ）
- 冒頭2秒で惹きつける『フック』→『本編（記事の要点）』→『CTA（続きはブログで等）』の構成
- 話し言葉。視聴者に語りかける口調
- 絵文字・記号・見出し・ナレーション以外の文字は出力しない（読み上げる文章のみ）
